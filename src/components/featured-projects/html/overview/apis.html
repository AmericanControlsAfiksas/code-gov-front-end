APIs

Some government agencies have APIs to allow members of the public and other agencies to access their data. Below are some open source government APIs that can be reused or serve as inspiration:

USA Spending API

This API is utilized by USAspending.gov to obtain all federal spending data which is open source and provided to the public as part of the DATA Act.

Known reuses: None

Code.gov API

This API is utilized by code.gov to obtain information about federal government open source projects and provided to the public as part of the Federal Source Code Policy.

Known reuses: None

Open FEC API

As the first RESTful API for the Federal Election Commission, this project aims to make campaign finance more accessible for journalists, academics, developers, and other transparency seekers.

Known reuses: None

18F API standards

This repository captures 18F's view of API best practices and standards.

Known reuses: None

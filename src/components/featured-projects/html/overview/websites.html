Websites

Most government agencies maintain at least one if not 10s or 100s of websites. Below are some open source government websites that can be reused or serve as inspiration:

United States Web Design System (USWDS)

The United States Web Design System (USWDS) is a library of open source UI components and a visual style guide for U.S. federal government websites. Both the USWDS website as well as USWDS itself are open source.

Known reuses: The USWDS team keeps their own inventory of the 100s of government websites that use the design system

code.gov

code.gov was created as part of the Federal Source Code Policy. The code.gov team evangelizes and follows open source best practices to develop code.gov.

Known reuses: None
